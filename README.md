# wdio_tests

WDIO tests repo is an example of WDIO tests implemented using PageObject pattern for learning purposes

## Installation

```bash
npm install
```

```Chai

npm install chai --save-dev

```

## Usage

### Launch tests:
```bash
npm run wdio
```

### Run eslint rules and fix errors if possible:
```bash
npm run lint
```
